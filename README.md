# 📝 Notes App

A sleek, intuitive, and responsive web application designed to help you quickly jot down thoughts, organize ideas, and manage daily tasks. Built entirely with vanilla web technologies, it features data persistence so you never lose your thoughts.

## ✨ Features

*   **Full CRUD Operations:** Create, read, update, and delete notes effortlessly.
*   **Persistent Storage:** Uses the browser's `localStorage` to keep your notes saved even after refreshing the page or closing the browser.
*   **Dynamic Search:** Instantly filter and find specific notes by title or content using a real-time search bar.
*   **Responsive Layout:** A clean, fluid grid system that adapts perfectly to mobile, tablet, and desktop screens.
*   **Timestamp Tracking:** Automatically records and displays the date or time each note was last modified.

## 🛠️ Tech Stack

*   **Structure:** HTML5
*   **Styling:** CSS3 (Flexbox/Grid, Custom Properties)
*   **Functionality:** JavaScript (ES6+, DOM Manipulation, LocalStorage API)
